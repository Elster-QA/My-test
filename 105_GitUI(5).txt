*************** Git UI***********************

Вызывается UI:
1) ПКМ в окне нашей локальной директории(Git GUI Here)
2) через cmd git gui&

***************ВНЕСЕНИЕ ИЗМЕНЕНИЙ в ФАЙЛ Git UI***********************
1) Открываем файл и дописываем строку new line GUI (СОХР.)
2) в UI нажим. на кнопку "Rescan"
3) Вместо ком.git add(перед комитом) мы нажимаем на иконку слева от фала
в окне "Unstaged" и файл переместится в окно "Staged Changes(Will Commit)" 
Т.Е. готов к комиту
...ЕСЛИ МЫ НЕ ХОТИМ ЭТО ДЕЛАТЬ ТО наж. на иконку слева от файла в окне 
"Staged Changes(Will Commit)" 
4) в окне "Commit message"  пиш. название нашего комита(прим."New Line for Git GUI")
5) жмем Commit
...в cmd можем проверить git status

...теперь можно СИНХРОНИЗИРОВАТЬ ДВА РЕПА. ЛОК. и УДАЛ.
6) наж. на Push(выбираем ветку на УДАЛ.РЕП.в которую будем заливать***) и Push

***ВАЖНО!!! МЫ ДОЛЖНЫ ЗАЛИВАТЬ В ТУ ВЕТКУ В КОТОРОЙ НАХОДИМСЯ НА ДАННЫЙ МОМЕТ!!!
...git branch(в cmd) МЫ УВИДЕМ ВЕТКУ В которой находимся и если хоти изменить ветку,
то в cmd git checkout имяветки
